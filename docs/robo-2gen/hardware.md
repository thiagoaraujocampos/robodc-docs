# Hardware - 2ª Geração

## Mudanças Relacionadas ao Robô de 1ª Geração

### Tabela Comparativa

| Componente | 1ª Geração | 2ª Geração | Melhoria |
|------------|------------|------------|----------|
| **Computador de Bordo** | Raspberry Pi 3/4 | NUC ou equivalente | Maior capacidade de processamento |
| **Câmera** | Câmera USB básica | Câmera de maior resolução | Melhor qualidade de imagem |
| **LIDAR** | LIDAR 2D padrão | LIDAR 2D/3D avançado | Maior alcance e precisão |
| **IMU** | IMU básica | IMU de 9 DOF | Maior precisão de orientação |
| **Bateria** | Padrão Li-Po | Li-Po de maior capacidade | Maior autonomia |
| **Motores** | Motores DC padrão | Motores com encoders de alta resolução | Maior precisão de controle |

## Novos Sensores e Atuadores

### Melhorias de Sensoriamento

A 2ª geração pode incluir sensores adicionais dependendo da configuração:

- **Sensores de profundidade (opcional)**: Câmeras RGB-D ou estéreo para mapeamento 3D
- **Sensores ultrasônicos (opcional)**: Detecção complementar de obstáculos
- **GPS para ambientes externos (futuro)**: Se houver expansão para operação externa

### Expressões Visuais

- **Display LED aprimorado**: Maior resolução para expressões faciais
- **Iluminação RGB**: Feedback visual de estado do robô

## Plataforma Robótica

### Base Atualizada

A base robótica da 2ª geração mantém o conceito de locomoção diferencial, mas com melhorias:

- Estrutura mais rígida para melhor estabilidade
- Melhor distribuição de peso
- Facilidade de acesso aos componentes internos
- Modularidade para facilitar substituição de peças

### Especificações Gerais

- **Dimensões**: Varia conforme configuração específica
- **Peso**: Aproximadamente 10-15 kg (varia com configuração)
- **Velocidade Máxima**: Até 0.5 m/s (configurável)
- **Autonomia**: 2-4 horas (depende do uso)

## Considerações de Montagem

### Layout Melhorado

O layout interno foi reorganizado para:

- Melhor ventilação de componentes eletrônicos
- Fácil acesso para manutenção
- Organização de cabos mais eficiente
- Espaço para futuros upgrades

### Acessibilidade

Melhorias na manutenibilidade:

- Painéis removíveis para acesso rápido
- Conexões padronizadas
- Documentação clara de cabeamento
- Etiquetas identificando componentes

### Proteção

Proteção aprimorada de componentes:

- Case protetora para computador de bordo
- Proteção de sensores contra impactos
- Isolamento de componentes elétricos
- Proteção contra curto-circuitos

## Considerações de Segurança

As mesmas considerações de segurança da 1ª geração se aplicam, com adições:

- Botão de emergência mais acessível
- Monitoramento de temperatura de componentes
- Sistema de desligamento automático em situações críticas
