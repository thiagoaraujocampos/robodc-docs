Visão Geral do Projeto

O RobôDC é um projeto de robótica móvel desenvolvido no Departamento de Computação (DC) da Universidade Federal de São Carlos (UFSCar) com o objetivo de criar uma plataforma robótica autônoma multifuncional.

O que é o RobôDC?​

O RobôDC é uma plataforma robótica móvel interdisciplinar, modular e anfitriã que serve como base para:

  • Atividades de ensino em disciplinas de robótica, sistemas embarcados e inteligência artificial
  • Projetos de pesquisa em navegação autônoma, percepção e controle
  • Iniciativas de extensão e interação com a comunidade
  • Função de robô anfitrião no Departamento de Computação

O projeto é pensado como uma plataforma viva e evolutiva, permitindo que diferentes grupos contribuam, integrem novas tecnologias e evoluam o sistema ao longo do tempo.

Histórico do Projeto​

O projeto RobôDC foi concebido no contexto do Laboratório de Robótica e Sistemas Inteligentes (LARIS) da UFSCar, evoluindo através de diferentes gerações:

1ª Geração​

A primeira geração focou em estabelecer a base do projeto:

  • Desenvolvimento da estrutura mecânica e integração de hardware
  • Implementação da pilha de software baseada em ROS 1 Noetic
  • Validação de conceitos de navegação autônoma
  • Criação da infraestrutura de desenvolvimento e documentação

2ª Geração​

A segunda geração representa uma evolução significativa:

  • Migração para ROS 2 Humble
  • Melhorias na arquitetura de hardware
  • Adoção de tecnologias mais modernas (Nav2, Behavior Trees)
  • Refinação da organização do código e documentação

Tecnologias Utilizadas​

Software​

  • ROS 1 Noetic (1ª geração) - Framework robótico
  • ROS 2 Humble (2ª geração) - Framework robótico de nova geração
  • Python e C++ - Linguagens de programação principais
  • Gazebo - Simulador 3D
  • RViz/RViz2 - Visualização e debugging
  • Nav2 - Stack de navegação (2ª geração)
  • Flask - API REST para controle remoto
  • Ionic/Angular - Aplicativo móvel

Hardware​

O hardware varia entre as gerações, mas tipicamente inclui:

  • LIDAR para mapeamento e detecção de obstáculos
  • Câmeras para visão computacional
  • IMU para odometria
  • Microcontroladores (Arduino/Raspberry Pi Pico)
  • Computador embarcado (Raspberry Pi/NUC)
  • Motores DC com encoders
  • Bateria e sistemas de alimentação

Colaboradores​

O projeto está aberto a contribuições de estudantes, pesquisadores e entusiastas de robótica.
